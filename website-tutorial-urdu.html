ضروری سافٹ ویئر انسٹال کریں

سب سے پہلے آپ کو ایک کوڈ ایڈیٹر چاہیے۔ سب سے مشہور اور مفت ایڈیٹر VS Code ہے۔ اسے code.visualstudio.com سے ڈاؤن لوڈ کریں۔ اس کے علاوہ گوگل کروم براؤزر بھی ضروری ہے۔

💡 ٹپ: VS Code مفت ہے اور ونڈوز، میک اور لینکس تینوں پر چلتا ہے۔
۲

پہلی HTML فائل بنائیں

VS Code کھولیں، ایک نئی فائل بنائیں اور اسے index.html نام سے محفوظ کریں۔ HTML وہ زبان ہے جس سے ویب صفحات کا ڈھانچہ بنتا ہے۔

index.html 
<!-- یہ ہے آپ کی پہلی HTML فائل -->
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>میری پہلی ویب سائٹ</title>
  </head>
  <body>
    <h1>السلام علیکم!</h1>
    <p>یہ میری پہلی ویب سائٹ ہے۔</p>
  </body>
</html>
🔎 یاد رکھیں: ہر HTML ٹیگ کو بند کرنا ضروری ہے، جیسے <h1> کو </h1> سے۔
۳

CSS سے ڈیزائن بنائیں

CSS (Cascading Style Sheets) سے آپ اپنی ویب سائٹ کو خوبصورت بناتے ہیں — رنگ، فونٹ، سائز سب CSS سے کنٹرول ہوتا ہے۔ ایک نئی فائل style.css نام سے بنائیں۔

style.css 
/* پس منظر اور فونٹ */
body {
  background-color: #f0f4ff;
  font-family: Arial, sans-serif;
  text-align: center;
}

/* بڑی سرخی */
h1 {
  color: #1a6ef5;
  font-size: 2.5rem;
  margin-top: 40px;
}

/* پیراگراف */
p {
  color: #374151;
  font-size: 1.1rem;
}

اب اسے HTML میں جوڑیں — <head> کے اندر یہ لائن ڈالیں:

index.html (head کے اندر) 
<link rel="stylesheet" href="style.css">
۴

JavaScript سے زندگی ڈالیں

JavaScript سے ویب سائٹ متحرک (interactive) بنتی ہے — بٹن دبانے پر کچھ ہونا، پیغام دکھانا وغیرہ۔ ایک فائل script.js بنائیں۔

script.js 
// بٹن دبانے پر پیغام دکھائیں
function سلام() {
  alert("السلام علیکم! آپ کی ویب سائٹ کام کر رہی ہے 🎉");
}

HTML میں بٹن اور JavaScript فائل اس طرح ڈالیں:

index.html (body کے اندر) 
<button onclick="سلام()">مجھے دبائیں</button>

<script src="script.js"></script>
۵

ویب سائٹ براؤزر میں کھولیں

اب index.html فائل پر ڈبل کلک کریں — وہ کروم براؤزر میں کھل جائے گی۔ آپ کی پہلی ویب سائٹ تیار ہے! کوئی بھی تبدیلی کریں، فائل محفوظ کریں (Ctrl + S) اور براؤزر ریفریش کریں (F5)۔

🚀 اگلا قدم: VS Code میں Live Server ایکسٹینشن انسٹال کریں — یہ ہر تبدیلی پر خود بخود ریفریش کرتی ہے۔
۶

ویب سائٹ آن لائن کریں

اپنی ویب سائٹ دنیا کو دکھانے کے لیے اسے GitHub Pages یا Netlify پر مفت میں ہوسٹ کریں۔ دونوں سروسز بالکل مفت ہیں اور شروع کرنے کے لیے بہترین ہیں۔

📌 طریقہ: github.com پر اکاؤنٹ بنائیں ← نئی repository بنائیں ← فائلیں اپ لوڈ کریں ← Settings میں Pages آن کریں۔ بس!
